

## Sobre
Esse é um repositório para testes chamado Projeto Antigo.
Repositório de exemplo / didático em Java, criado para ser utilizado em aulas sobre Git e GitHub.
Este projeto é simples e serve apenas como material didático para demonstração de controle de versão com Git e GitHub.  

## Pré-requisitos

- Java instalado 
- Git instalado para controle de versão  
- IDE de sua preferência para compilar/executar o código  

## Como executar

1. Clone o repositório:
   git clone https://github.com/otiliadb/ProjetoAntigo.git
2. Importe o projeto para a IDE escolhida


## Boas práticas com Git / GitHub

- Crie commits pequenos e com mensagens claras
- Use branches diferentes para funcionalidades ou exercícios distintos
- Sempre sincronize com o repositório remoto antes de iniciar mudanças
